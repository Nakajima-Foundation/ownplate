# 【新機能】お店独自の割引キャンペーンが設定できるようになりました

こんにちは。OwnPlate開発チームです。

この度、店舗様がお客様向けに独自の割引キャンペーンを設定できる新機能をリリースしました。「○○円以上で△△円引き」といった柔軟な販促施策が、管理画面から簡単に設定できるようになります。

## どんな機能?

各店舗が独自の値引きルールを設定できる機能です。

- **金額に応じた割引**:「3000円以上で300円引き」など
- **割引率での設定**:「10%OFF」など
- **期間限定キャンペーン**:開始日時・終了日時の設定が可能
- **決済方法の指定**:事前カード決済限定などの条件設定
- **利用回数制限**:初回のみ、または無制限

これにより、リピーター獲得や客単価アップなど、店舗様の戦略に合わせた販促が可能になります。

## 管理画面での設定方法

### 1. 割引の新規作成

管理画面から「新規」ボタンをクリックして、割引を作成します。

![](./images/discounts_management_top.png)

### 2. 設定項目

以下の項目を設定できます:

![](./images/discounts_management_new.png)

| 項目 | 内容 |
|------|------|
| **名前** | キャンペーン名(お客様にも表示されます) |
| **割引の有効化** | ON/OFFで即座に切り替え可能 |
| **適用期間** | 期間限定にする場合は開始・終了日時を設定 |
| **最低利用金額** | 「○○円以上で適用」の条件設定 |
| **利用回数制限** | 1回のみ or 無制限 |
| **割引方法** | 金額割引(¥100) or 割引率(10%) |
| **決済方法制限** | カード決済限定などの制限が可能 |

### 3. 複数割引の扱い

**重要なポイント:**
- 複数の割引を設定できますが、1回の注文で適用されるのは**1つのみ**
- 条件を満たす割引が複数ある場合、**最大の割引額**が自動適用されます
- 金額割引と割引率を同時期に混在させないことを推奨します

### 4. 利用履歴の確認

どのお客様が、いつ、いくら割引を利用したかを確認できます。

| 表示項目 | 内容 |
|---------|------|
| 利用者 | ユーザー情報へのリンク |
| 注文番号 | 注文詳細へのリンク |
| 注文日 | 日時 |
| 合計金額 | 割引前の金額 |
| 割引 | 適用された割引額 |

これにより、キャンペーンの効果測定も簡単に行えます。

## お客様側の表示

### 店舗トップページ

カートが空の状態では、フローティングボタンでキャンペーンをアピールします。

![](./images/discounts_user_top.png)

ボタンをタップすると、現在実施中のキャンペーン一覧が表示されます。

![](./images/discounts_user_top_list.png)

### カート画面での表示

商品を選択していくと、「あと○○円で割引適用」という形で、お客様に次の目標金額が表示されます。

**例: 1000円と3000円の2段階割引を設定した場合**

- カート内が1000円未満のとき
![](./images/discounts_user_cart_1000.png)

- 1000円〜3000円未満のとき
![](./images/discounts_user_cart_2000.png)

- 3000円以上のとき(最大割引適用)
![](./images/discounts_user_cart_3000.png)

このように、段階的に割引が適用されることで、お客様の購買意欲を高める効果が期待できます。

### 注文確認画面

最終的に適用されたキャンペーン名と割引額が明確に表示されます。

![](./images/discounts_user_confirm.png)

合計金額は値引き後の金額が表示されるので、お客様も安心してご注文いただけます。

### 注文完了画面

注文完了後も、適用されたキャンペーン情報が確認できます。

![](./images/discounts_user_fin.png)

## 活用シーン

この機能は、様々なシーンで活用できます:

### ケース1: 客単価アップ施策
「3000円以上で300円引き」のように、少し背伸びすれば届く金額を設定することで、お客様に「もう一品」を促すことができます。

### ケース2: 初回限定キャンペーン
利用回数制限を「1回のみ」に設定し、新規のお客様の獲得を目指します。

### ケース3: 期間限定セール
週末限定や、お店の記念日など、特定期間のみ大幅割引を実施できます。

### ケース4: 決済方法の誘導
「事前カード決済で10%OFF」など、オペレーション効率化につながる決済方法への誘導も可能です。

## まとめ

割引機能を活用することで、店舗様の販促施策の幅が大きく広がります。

**ポイント**
- 管理画面から簡単に設定・ON/OFF切り替えが可能
- お客様には自動で最大の割引が適用される
- 利用履歴で効果測定ができる
- 段階的な割引設定で購買意欲を高められる
